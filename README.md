The [official website](https://lynxjs.haskell-miso.org) for the `miso-lynx` project.

## Setup :crane:

Install the dependencies:

```bash
bun install
```

## Get started

Start the dev server:

```bash
bun run dev
```

Build the website for production:

```bash
bun run build
```

Preview the production build locally:

```bash
bun run preview
```
